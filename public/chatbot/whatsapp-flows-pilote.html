⚠ Règles Meta critiques
  • Écran 1 = JSON only · tout vient de flow_action_data
  • Pas de INIT call côté Meta · le bouton template envoie tout
  • Variables ${'$'}{data.x} seules · TextCaption + TextBody sur 2 lignes
  • Max 5-8 écrans par flow · routing_model statique OK
  • Dernier écran terminal: true + action complete
  • Chiffrement AES-GCM côté webhook · noté en annotation, pas dans le design

  • Bouton primaire vert WA · secondaire outline
  • Numéro écran X/Y en gris top-right
